Data Structure and Programming

  • 1/4/2023
  • 2-minute read

Good Java
Data Structure is a fundamental but vital for any software engineer. here are some important topics that I have learnt.


Important Topics Covered :
Big-O Notation gives a unified way to evaluate the efficiency of algorithms.
Stacks, Queues, and Linked Lists handle data in simple, linear ways (LIFO, FIFO, and node-based structures).
Binary Trees and their Traversals (preorder, inorder, postorder) provide hierarchical organization and flexible data access.
Binary Search Trees (BSTs) use sorted ordering for efficient search, insertion, and deletion.
AVL Trees and 2–3 Trees are self-balancing BSTs ensuring consistently fast operations.
Heaps/Priority Queues maintain quick access to the maximum or minimum element for scheduling or sorting tasks.
Graphs and Traversals (DFS/BFS) model complex relationships for networking, routing, and discovery problems.
Hashing (separate chaining/open addressing) uses a hash function to achieve near-constant lookups.
Union-Find (Disjoint Set) efficiently merges and finds connected components, key in algorithms like Kruskal’s.
Kruskal’s Algorithm builds minimum spanning trees by repeatedly adding the smallest edges that don’t form cycles.
Selection Algorithm (e.g., Quickselect) locates the k-th smallest or largest element without fully sorting.
